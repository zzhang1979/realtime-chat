# Realtime Chat App

An enhanced realtime chat application built with Node.js, Socket.IO, and Express.

## Features
- ✅ User nicknames for personal identification
- ✅ Real-time messaging with Socket.IO
- ✅ Message timestamps
- ✅ User join/leave notifications
- ✅ Responsive, modern UI
- ✅ Private messaging foundation (easy to extend)
- ✅ Cross-browser compatible

## Technology Stack
- **Backend**: Node.js with Express and Socket.IO
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Realtime**: Socket.IO WebSocket implementation

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start` or `node index.js`
4. Open browser to: http://localhost:3000
5. Set your nickname and start chatting!

## Features to Explore
- Open multiple browser windows to test realtime communication
- Try setting different nicknames
- Observe join/leave notifications when users connect/disconnect
- Messages appear with timestamps and user identification
